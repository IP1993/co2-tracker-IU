# CO₂ Tracker – IU Webprojekt

Dieses Repository enthält den **CO₂‑Tracker**, ein fiktives Webprojekt zur Darstellung und Filterung von CO₂‑Emissionsdaten.  
Das Projekt wurde im Rahmen des IU‑Moduls  
**IPWA01‑01 „Programmierung von Webanwendungsoberflächen“**  
entwickelt.

---

## Ziel des Projekts

Ziel dieses Projekts ist es, eine benutzerfreundliche Weboberfläche zu erstellen, die:

- fiktive CO₂‑Emissionsdaten verschiedener Länder und Unternehmen anzeigt  
- einfache Filterfunktionen bereitstellt  
- ein responsives Layout nutzt  
- grundlegende Sicherheitsmaßnahmen für Eingabefelder umsetzt  
- modernes, übersichtliches Webdesign verwendet  

Alle Daten sind **fiktiv** und dienen ausschließlich der Demonstration.

## Funktionen

### ✔ CO₂‑Datentabelle
- Anzeige fiktiver Emissionsdaten  
- Daten aus mehreren Kontinenten, Ländern und Unternehmen  

### ✔ Filteroptionen
Filterbar nach:
- Land  
- Unternehmen  
- maximalen Emissionen  

### ✔ Kontaktformular
- rein lokale Verarbeitung  
- einfache Eingabesäuberung (Sanitization)  
- keine Speicherung oder Weitergabe von Daten  

### ✔ Dark Mode
- per Button umschaltbar  
- auf allen Seiten verfügbar
